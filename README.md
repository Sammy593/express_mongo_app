
# Template Express + MVC + API

## Run Locally

Clone the project

```bash
  git clone https://github.com/Sammy593/express_app.git
```

Go to the project directory

```bash
  cd express_app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
